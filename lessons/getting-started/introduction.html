Introduction

Welcome to your Generic Notes Repository! 🎉

This site is powered by Next.js and generates a beautiful, static documentation site from Markdown files. It's the perfect place to drop all your notes, tutorials, and snippets.

How it works

  1. This entire site is generated from the lessons/ folder in your repository.
  2. Every folder inside lessons/ becomes a Section in the sidebar.
  3. Every Markdown file inside a folder becomes a Page.
