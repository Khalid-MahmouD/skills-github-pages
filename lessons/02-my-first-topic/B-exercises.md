# Exercises

### Try this yourself!

To prove this works, try deleting these files and adding your own!

1. Delete `01-getting-started` and `02-my-first-topic` from the `lessons/` folder.
2. Create a new folder called `01-react-basics`.
3. Create a `meta.json` inside that folder.
4. Add your first `A-components.md` file.

Commit the changes, push them to GitHub, and watch the GitHub Actions workflow automatically rebuild your site with your new content!
